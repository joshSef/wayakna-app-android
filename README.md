# Wayakna Catalog App

Aplicación móvil MVP desarrollada en Android (Jetpack Compose)
para presentar paquetes de consultoría en Odoo y Correobox.

## Funcionalidades
- Pantalla Home institucional
- Listado de paquetes
- Detalle de paquete
- Pantalla de contacto
- Navegación entre pantallas
- Botón de contacto por WhatsApp

## Arquitectura
- Kotlin
- Jetpack Compose
- Navigation Compose
- Patrón simple Repository
- Gradle Wrapper 8.7

## Estado
Aplicación Android en evolución para catálogo comercial de servicios
